Help

How to query data?

  • Table: Select Database table, no need to select if there is only one table in database.
  • Data: Display data in selected period.
  • Fields: There are 16 photovoltaic panels connecting to a combiner box, select the panel number to view its data.
  • Click the "Query" button to view the selected data.

Data Analysis

  • Exploratory Data Visualization Chart: Display the raw data of selected photovoltaic panels.
  • Statistical Analysis Chart: Display the standard deviation analysis of the data from the selected panels.
