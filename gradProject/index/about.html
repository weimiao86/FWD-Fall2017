Renewable Engery Analysis Lift

The Renewable Engery Analysis Lift (REAL) is a data visualization platform, it is developed for Concord New Energy (CNE) in their photovoltaic panels fault detection research. This tool provides the drectly data query from CEN's database and generating the statistical analysis chart and graph for researchers to analysis and predict photovoltaic panels fault.
